二:windows快捷键：
	ctrl + tab			    切换窗口
	shift + tab			    页面内切换焦点
	windows +	--> 		  分屏
	win键	+ ←/→			    使该应用缩至一半,并能调动左右位置
                  		在上一步的基础上,同时按住win键+↑/↓,能实现右上角四分一窗口显示
	windows	+ ↑/↓		    最大/小化当前页面
	ctrl + w 			      关闭当前页面
	alt	+ f4			      关闭当前页面
	ctrl + m			      确定
  window + Q          搜索
	ctrl + f4           清除缓存


三:sublime快捷键
	ctrl + L		选中本行
	end					行尾
	home				行首
	alt + f3		选中相同的单词进行编辑
	alt					鼠标从界面移动到菜单栏
  ctrl + D  	先选中某个单词,再按快捷键可以往下继续选中和它相同的
  ctrl + g 		快速定位到某一行



四:emmet语法
	ul+tab  			            快速添加私有前缀
	p*5     			            会得到5个p标签
	ul>li*5 			            一个ul标签,其下再包含5个li标签 					 乘法
	nav>ul>li 			          一个nav包含一个ul,ul又包含一个li 				 后代
	div+h3+p*3 			          一个div后接着一个h3再接着3个p 					 兄弟
	div>dl>(dt+dd*3) 	        div下包含dl,dl下包含dt和3个dd 					 缩写
	ul>li.classname$*5 	      ul下5个li标签,标签带有类属性名,且属性名依次递增   $ 递增符
	#idname 			            带有id名的div
	p.classname1.classname2   p标签同时带有两个类名
	ul>.classname 		        ul下有个带类名的li标签
其他还有很多自行百度
